# LEARN-HTML-CSS

Welcome to the **LEARN-HTML-CSS** repository! This project is a collection of my work and practice files as I build my foundational skills in web development using HTML and CSS.

## 🌐 Live Demo

You can view the live deployment of this project here:  
**[https://learn-html-css-1.onrender.com](https://learn-html-css-1.onrender.com)**

## 🛠️ Technologies Used

* **HTML5:** For structuring the web content.
* **CSS3:** For styling, layout, and responsive design.
* **Render:** Used for hosting and deploying the static site.

## 🚀 How to Run Locally

If you want to run this project on your local machine, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rishumour/LEARN-HTML-CSS.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd LEARN-HTML-CSS
    ```
3.  **Open the file:**
    Simply double-click the `index.html` file to open it in your default web browser, or use an extension like **Live Server** in VS Code for a better development experience.
